# FraudGuard AI

FraudGuard AI is an AI-powered banking fraud detection system built with Java Spring Boot, Python FastAPI, React, Material UI, MySQL, and Machine Learning.

The system analyzes financial transactions, predicts suspicious or fraudulent activity, assigns fraud risk scores, and provides an admin dashboard for fraud investigation.

## Tech Stack

- Java Spring Boot
- Spring Security + JWT
- Python FastAPI
- Scikit-learn
- React
- Material UI
- MySQL

## Main Modules

### Backend

Handles authentication, users, transactions, fraud cases, audit logs, and communication with the AI service.

### AI Service

Loads the trained machine learning model and exposes prediction endpoints for fraud detection.

### Frontend

Provides the admin dashboard, transaction upload page, fraud alerts page, and analytics charts.

### Database

Stores users, transactions, fraud predictions, fraud cases, and audit logs.
